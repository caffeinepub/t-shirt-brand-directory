# Specification

## Summary
**Goal:** Create a t-shirt brand aggregator website that displays multiple t-shirt brands with their website links and price ranges, and allows users to add new brands.

**Planned changes:**
- Build backend data model to store t-shirt brand information (name, website URL, price range)
- Implement backend methods to add, retrieve, and list brands
- Create frontend page displaying all brands in a responsive grid/list layout
- Display brand name, clickable website link, and price range for each brand
- Add form to submit new t-shirt brands with validation
- Apply modern, fashion-focused visual design with consistent styling

**User-visible outcome:** Users can browse a curated list of t-shirt brands with pricing information and links to their websites, and contribute new brands through an add form.
